# Hotel-Management-System
Python hotel billing system featuring menu display, multiple item ordering, quantity selection, and automatic receipt generation.

## Features

- Display menu with prices
- Order multiple items
- Select quantities
- Automatic bill calculation
- Multiple order support using loops
- Itemized receipt generation

## Technologies Used

- Python
- Dictionaries
- Loops
- Conditional Statements

## Sample Output

```
========== BILL ==========
Pizza x 2 = Rs240
Coffee x 3 = Rs120
--------------------------
Total Amount: Rs360
```

## How to Run

1. Clone the repository
2. Open terminal
3. Run:

```bash
python hotelmenu.py
```
